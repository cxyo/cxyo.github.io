## 四、单均线策略

### 4.1单均线策略原理

#### 4.1.1均线原理

单均线策略的核心指标是简单移动平均线，简称均线。

均线由美国投资专家葛兰威尔所创立，由道氏股价分析理论的“三种趋势说”演变而来，从数字的变动中去预测股价未来短期、中期、长期的变动方向，为投资决策提供依据。

按照周期长短划分，均线可以分为短期均线、中期均线、长期均线三种基本类型。

1）短期均线：5、7、10，用于预测短期走势，MA5 和 MA10 又称为短期监测线；
2）中期均线：20、30、60，用于预测中期走势，MA20 和 MA30 又称为警戒线， MA60 则称之为生死线；
3）长期均线：120、250，用于长期走势，MA120 又称为确认线，MA250 则通常被看做反转线，又称为牛熊分界线。

##### 均线计算

计算均线需要先设置均线周期，将均线周期设置为15日，就能进入下面的计算：

将第 1-15 个交易日的收盘价求和，除以 15，就能得到第一个均价；

向后移动一个交易日，再将第 2-16 个交易日的收盘价求和，除以 15，得到第二个均价；

重复以上步骤，将得到的所有均价连成一条线，就生成了期间的 15 日均线。

#### 4.1.2单均线策略核心内容

单均线策略核心内容就是**「简单判断股票价格的趋势以及买入、卖出的时间点」**。

单均线的使用核心：在当前交易日，股票前天 15 日均价**大于**昨天 15 日均价，昨天 15 日均价**小于**今日 15 日均价，这时均线会出现**「向上的拐点」**，则大概率认为股票价格在未来一段时间内上涨；

而当该股票前天 15 日均价**小于**昨天 15 日均价，昨天 15 日均价**大于**今日 15 日均价，均线出现**「向下的拐点」**，则大概率在未来一段时间内下跌。

总结一下：在当前交易日，当某股票前天 15 日均价**大于**昨天 15 日均价，昨天 15 日均价**小于**今日 15 日均价，指挥计算机**买入**股票；而当该股票前天 15 日均价**小于**昨天 15 日均价，昨天 15 日均价**大于**今日 15 日均价，**卖出**股票。

## 五、双均线策略

### 5.1双均线策略原理

#### 5.1.1、短期均线和长期均线

与单均线策略类似，双均线策略的核心指标也是均线，是一种经典的趋势跟踪策略，经常会用来实操捕捉市场行情中的一段大趋势。

不同之处在于双均线策略会用到两条均线，一条叫做`短期均线`，另一条叫做`长期均线`。

短期均线和长期均线是相对的，一般来讲，均线周期较短的为短期均线，均线周期较长的为长期均线。

#### 5.1.2、双均线核心内容

双均线策略的思想也不复杂，它是根据**长短周期的价格的移动平均线之间的关系确定买卖点**。

其中，短周期均线「从下向上」突破长周期均线时，意味着当前时间段具有上涨趋势，突破点也就是常说的「金叉」，这是股票**「多头信号」**。

当短周期均线「从上向下」跌破长周期均线时，则意味着当前时间段具有下跌趋势，跌破点也就是常说的「死叉」，这是**「空头信号」**。

## 二、移动平均成本法的魅力

### 2.1移动平均成本法的定义

当定投基金标的净值低于平均持有成本时，增加投资金额去购买更多的基金份额；当定投基金标的净值高于平均持有成本时，减少投资金额去购买更少的基金份额。

![image-20220621144851232](https://cdn.jsdelivr.net/gh/cxyo/hexo/imgimage-20220621144851232.png)

##### **`偏离度 = (T-1日基金净值 - 基金单位持有平均成本) / 基金单位持有平均成本 * 100`。**

![image-20220621145012442](https://cdn.jsdelivr.net/gh/cxyo/hexo/imgimage-20220621145012442.png)

基金净值，等于基金的总资产减去总负债后的余额再除以基金全部发行的单位份额总数。简单理解，就是基金在申购和赎回时显示的价格。

而基金单位持有平均成本，其实就和上节课学的均线概念一致，它代表基金持有的平均价格。

所以，偏离度其实就表示定投基金的净值与平均持有成本的比较。一般将它的结果设定为`± 10%`，可以理解为大于 10% 减少投入，小于 10% 增加投入。

### 2.2移动平均成本法的实用技巧

最理想的状态就是在投资标的**低价**时买入很多，等它上涨；在**高价**时，少量买入或不买入，并且阶段卖出。这样，就可以赚取更多的超额收益。

如何做才能在基金低价时多买一些份额，在基金高价的时候，少买或者不买一些份额？

实际操作就是从定期定额投入金额的方法转变为定期但不定额的投入金额。

这种玩法其实就是**移动平均成本法**。它更多是利用正确的量化策略来辅助你进行定投决策，并且你也可以通过真实投资情况来调整参数条件与实际投资资金。

### 2.3移动平均成本法的100%胜率

用一段时间的上证综指（1991 年 1 月 - 2014 年 4 月）进行测算，测算结果发现：

只要能坚持连续三年定投，每月第一个交易日为扣款日。每期正常投资金额为 1000 元。利用移动平均成本法的定期不定额，收益率将会以 **100%** 的胜率战胜普通定投的收益率！

```python
    """指标"""
    # 定投日期, 每月9号定投
    context.buy_day = 9

    context.buy_money = 5000

    context.up_bound = 1.1  
    # 偏离下界限
    context.down_bound = 0.9
    # 向上突破上界限时的定投倍数  
    context.up_multiple = 0.5
    # 向下突破下界限时的定投倍数  
    context.down_multiple = 1.5

    """择时"""
    # 获取当前交易日的日期
    current_date = context.data.datetime.date(0)
    # 获取上一个交易日的日期
    previous_date = context.data.datetime.date(-1)
    # 判断当前交易日是否是定投日期
    if context.buy_day == int(str(current_date)[-2:]) or int(str(previous_date)[-2:]) < context.buy_day < int(str(current_date)[-2:]):
        # 获取标的的仓位
        position = context.getposition(context.data).size
        
        # 判断是否持仓
        if position > 0:
            # 如果持仓，获取平均持有成本
            per_value = context.getposition(context.data).price
        else:

            per_value = context.data.close
        # 获取定投的费率
        taker = context.get_taker(context.buy_money)
        # 基于定投金额和费率计算申购份额
        size = context.buy_money / (context.data.close * (1 + taker))
        
        # 计算当前标的价格与平均持有成本的比值
        ratio = context.data.close[-1] / per_value
        
        # 判断是否偏离下界限
        if ratio < context.down_bound:
            # 当当前价格低于平均持有成本时，则多买
            size = size * context.down_multiple
            
        # 判断是否偏离上界限
        if ratio >= context.up_bound:
            # 当当前价格高于平均持有成本时，则少买
            size = size * context.up_multiple
            
		# 发送申购指令
        context.order = context.buy(data=context.data, size=size)
```

这种方法被叫做买入并持有，同时我们也把它称为**Buy and Hold**。

此次定投回测，如果我们在策略开始时运用Buy and Hold方法来定投，截止回测结束，收益率为 3.64%，无法达到移动平均成本法的收益结果。

可以这样理解，当你利用量化交易的移动平均成本法执行定投策略时，就是一种偏**主动性**的动态交易，它能指导我们实现低点多买、高点少买的理想状态。

而 Buy and Hold 一般是常规定投方法，是一种偏**被动性**的稳定交易。每月不论行情好坏，只要持续买，也可以用时间换取空间获益（相对理想状态）。

利用这两个收益率的对比可以很直接地让我们判断当前策略的有效性。

### 2.6移动平均成本法的策略分析

**移动平均成本法：基金净值低于平均成本时，增加定投金额；基金净值高于平均成本时，减少定投金额，使平均成本在市场下跌时较快下降，在市场上升时较慢上升。**

移动平均法的策略设定：
`偏离度 = (T-1 日基金净值 - 基金单位持有平均成本) / 基金单位持有平均成本 * 100%`
设定为 ±10%，大于 10% 减少投入， 小于 10% 增加投入。

标注一下关键字：根据扣款前一交易日（T-1 日）的基金净值和平均成本的差距（偏离度）来确定每期定投金额。

所以，理论上而言，偏离度、定投金额，基金净值，甚至定投时间都会在一定程度上影响策略结果。

### 3.3量化投资组合-动量增强策略

#### 3.3.1、动量增强策略的简单概念

**动量效应**是指股票的收益率有延续原来的运动方向的趋势，即过去一段时间收益率较高的股票，在未来依旧会取得高于平均的收益率。

具体表现在股票市场中，是指那些过去表现好的股票在未来依然有概率完成较好的表现，因此不断选择前期表现好的股票即可获得超额收益，即动量选股策略。

#### 3.3.2、动量增强策略的可行性

简单理解，动量增强策略的核心就是最大限度的发挥策略的效力，即集中选出并持有最优的股票以及不断的汰弱留强。其潜在的优势是盈利空间巨大，劣势是盈利机会较少。

#### 3.3.3、动量增强策略-实践才是硬道理

1）移动平均时间窗口为 15；
2）用于购买股票的资金比例为 0.9；
3）需要考虑收益率的时间窗口为 50；
4）截面收益率的阈值为 0.2；

```python
def indicators(context):
    """指标"""
    # 移动平均时间窗口
    context.period = 15
    # 设置用于购买股票的资金比例
    context.max_value_percent = 0.9  
    # 设置需要考虑收益率的时间窗口
    context.look_back_days = 50  
    # 设置截面收益率的阈值
    context.threshold = 0.2

    # 创建两个空字典
    context.sma = dict()
    context.rate = dict()
    for data in context.datas:
        # 计算所有股票的15日的移动平均价，存储到字典context.sma中
        context.sma[data._name] = SMA(data.close, period=context.period)
        # 计算所有股票的截面收益率，存储到字典context.rate中
        context.rate[data._name] = PctChange(data.close, period=context.look_back_days)

        
        
def choose_stock(context):
    """标的"""
    # 股票组合的代码赋值给context.symbol_list
    context.symbol_list = ['600048.XSHG', '601010.XSHG', '600663.XSHG','600007.XSHG', '600185.XSHG']
    # 设置综合指数作为基准
    context.benchmark = "600048.XSHG"

    # 创建一个空列表
    rate_list = []
    # 遍历所有股票
    for data in context.datas:
        # 获取每只股票的截面收益率，存储到列表rate_list中
        rate = context.rate[data._name][0]
        if rate > context.threshold:
            rate_list.append([data._name, rate])

    # 按照收益率进行降序排序，并获取指定数量的股票池
    sorted_rate = sorted(rate_list, key=lambda x: x[1], reverse=True)
    context.stock_pool = [i[0] for i in sorted_rate]
    context.pool_size = len(context.stock_pool)

        # 遍历所有股票
        for data in context.datas:
            # 获取每只股票的仓位
            position = context.getposition(data).size
            # 如果该股票在股票池中，且未持仓，且当前交易日收盘价大于其15日均价,出现买入信号
            if not position and data._name in context.stock_pool and data.close[0] > context.sma[data._name][0] :
                # 计算该股票的买入数量
                size = int(p_value / 100 / data.close[0]) * 100
                # 发送买入指令
                context.buy(data=data, size=size)
                
                
def control_risk(context):
    """风控"""
    # 遍历所有股票
    for data in context.datas:
        # 获取每只股票的仓位
        position = context.getposition(data).size
        # 如果该股票已持仓，但不在股票池中，或当前交易日收盘价小于其15日均价,出现卖出信号
        if position != 0 and data._name not in context.stock_pool or context.sma[data._name][0] > data.close[0]:
            # 平仓
            context.close(data=data)
```
